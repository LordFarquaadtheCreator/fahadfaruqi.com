The gallery metaphor is useful because it forces restraint. A good room does not explain every object. It gives each object enough air to become legible.

Product surfaces can do the same. When typography, spacing, and hierarchy are allowed to breathe, the interface starts to feel more confident.

```ts
const surface = compose(content, rhythm, restraint);
```

The point is not minimalism for its own sake. The point is to reduce noise until the useful signal becomes unmistakable.
